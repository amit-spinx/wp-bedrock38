# WP Bedrock38

WordPress project using the **Bedrock** structure.

## Layout
- `web/app/` — wp-content (themes, plugins, mu-plugins, uploads)
- `web/wp/` — WordPress core (installed via Composer)
- `config/` — environment configuration
- `.env` — secrets & DB credentials (copy from `.env.example`)
